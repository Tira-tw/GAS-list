# 加油模擬工具網站

## 介紹
本網站為免費的加油模擬工具，提供用戶模擬加油站收據的輸入及條碼產生，僅供參考與學習用途，非正式交易系統，並不具法律效力。

## 功能
- 輸入加油類別、加多少（可輸入金額或「加滿」、「跳停」等文字）
- 可輸入統編、載具、會員資訊（可空白）
- 顯示載具與會員條碼（CODE128）
- 固定顯示作者資訊
- 預覽結果彈窗
- 使用條款與免責聲明強制同意彈窗
- 連結至完整使用條款獨立頁面

## 使用說明
1. 開啟 `index.html` 網頁
2. 首次使用時，需先閱讀並同意使用條款與免責聲明
3. 填寫加油模擬表單，送出後會顯示預覽結果及條碼
4. 連結可查看完整使用條款
